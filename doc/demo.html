Getting started

          
This is a full Ficl system running in your browser.
            At the top right you will see the state of the parameter stack.
            Bottom right are some simulated LEDS.
            If you are new to Forth, please follow along with this tutorial in a split view.
            You can find more information on Ficl here.
          

          
Demo words

          

            
!led  ( n -- ) 

            
Control simulated LEDs with the low 8 bits of n

            
ms  ( ms -- ) 

            
Insert delay in milliseconds 

            
yield  ( -- ) 

            
Give the browser a chance to refresh the display
